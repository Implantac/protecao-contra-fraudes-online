# Proteção Contra Fraudes Online

## Project Overview
"Proteção Contra Fraudes Online" is a web application designed to help users identify and report online fraud. It provides a dashboard with alerts on current scams, a search feature to look up companies or CNPJ numbers, educational resources on fraud prevention, and a platform for users to submit anonymous reports of suspicious activities.

## Installation
To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd <repository-directory>
   ```

2. **Open the `index.html` in your preferred web browser:**
   Simply double-click the `index.html` file or open it within your browser.

## Usage
Once the application is running, users can navigate through the following sections:
- **Dashboard:** Displays current alerts for high-profile scams and statistics regarding reported fraud.
- **Search:** Offers the ability to search for a company using its name or CNPJ.
- **Area Educativa:** Contains articles and videos educating users on how to avoid fraud.
- **Denúncia Anônima:** A form that allows users to anonymously report scams or suspicious activities.

## Features
- Real-time dashboard showing the latest fraud alerts.
- Easy search for companies and CNPJ.
- Educational articles and videos on identifying and avoiding fraud.
- Anonymous reporting form for users to disclose suspicious activities.

## Dependencies
This project relies on the following external resources:
- [Tailwind CSS](https://tailwindcss.com/) - A utility-first CSS framework for styling.
- [Font Awesome](https://fontawesome.com/) - Icon toolkit to enhance the interface.

No additional packages are required as this is a static HTML/CSS/JS application.
